Navigating IPCC Working Group I report

Naviguer dans les rapports du GIEC

The IPCC WGI report is a 3900-page summary of state-of-the-art scientific knowledge.
This app helps you navigate it, understand and learn from it.

Ce site vous aide à y naviguer, à le comprendre et à apprendre.

Downloading complete report

Téléchargement du rapport complet

Afin de pouvoir naviguer dans cette partie du rapport, cette application doit télécharger un fichier d'environ 80Mo.
Assurez-vous que cela n'engendre pas de surcoût pour votre abonnement Internet.
Ce téléchargement ne se fera qu'une fois, sauf si vous rechargez la page ou quittez le navigateur.

Si vous disposez déjà du fichier PDF complet du rapport du GIEC sur votre ordinateur (IPCC_AR6_WGI_Full_Report.pdf), vous pouvez l'utiliser directement en cliquant sur "Ouvrir en local" ci-dessous. Cela sera plus rapide, ou économisera votre forfait de données.

Cet outil n'est pas fonctionnel sur smartphone, seulement sur ordinateur.

Before allowing you to navigate through this part of the report, this app needs to download a 80MB file.
Please ensure your internet provided won't charge you for this amount of data.
This download will be done only one time, unless you reload the page or quit the browser.

If you already have the complete IPCC PDF report on your computer (IPCC_AR6_WGI_Full_Report.pdf), you can use it directly here by clicking "Open local" below. This will be faster and save bandwidth.

Navigating the PDF is heavyweight and is not made for smartphones, only for desktop computers.
